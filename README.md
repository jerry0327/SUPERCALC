# SUPERCALC 超級計算機

這是一個手機也能透過 GitHub Actions 打包成 APK 的 Android 測試專案。

## 功能

- 基本四則運算
- 科學運算：sin、cos、tan、log、ln、平方、次方、平方根、π、e、階乘、括號
- DEG / RAD 角度模式
- 計算歷史紀錄
- 3D Plot 視覺化示範：z = sin(x) * cos(y)、z = sin(r) / r、z = x² - y²、z = cos(x) + sin(y)
- 單位換算：長度、重量、溫度
- BMI 計算
- 貸款月付試算
- 折扣與稅率試算
- 深色科技風格 UI
- 隱藏彩蛋：貪吃蛇小遊戲

## 如何打包 APK

1. 打開 GitHub repo 的 Actions 頁面。
2. 選擇 `Build Android APK`。
3. 點 `Run workflow`。
4. 成功後下載 `supercalc-debug-apk` artifact。
5. 解壓縮後安裝 APK。

## 彩蛋

進入工具箱後點擊「貪吃蛇彩蛋」卡片即可遊玩。
